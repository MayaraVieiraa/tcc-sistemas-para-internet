## SIATRAB – Sistema de Apoio ao Trabalhador para Conferência de Cálculos e Direitos Trabalhistas

Este repositório contém o código-fonte e a documentação do sistema **SIATRAB**, desenvolvido como Trabalho de Conclusão de Curso no curso de **Tecnologia em Sistemas para Internet**.

O projeto tem como objetivo apoiar trabalhadores na compreensão de seus direitos trabalhistas e na conferência de cálculos relacionados a verbas trabalhistas, por meio de uma plataforma digital de caráter educacional e informativo.

### Finalidade do projeto

O sistema possui finalidade exclusivamente acadêmica e educacional, não substituindo a atuação de profissionais especializados, como advogados, contadores ou órgãos oficiais.

### Contexto acadêmico

Projeto desenvolvido no âmbito do Trabalho de Conclusão de Curso, como requisito parcial para obtenção do grau de tecnólogo em Tecnologia em Sistemas para Internet.
